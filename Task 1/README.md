# Mall Customer Segmentation using K-Means Clustering

## Project Overview

This project segments mall customers into different groups using the K-Means Clustering algorithm.

## Dataset

Mall_Customers.csv

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Steps

1. Load Dataset
2. Data Cleaning
3. Elbow Method
4. Train K-Means
5. Visualize Clusters

## Results

The optimal number of clusters was 5.
